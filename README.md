# Laravel-Ecommerce-Project

You will find all codes in master branch.

PHP version 7.3.*

Laravel version 8.2*

set up database by importing ecommerce_project.sql from database folder. update your .env file as per your db connection. might need to set up mail and store credentials as per yours. do composer update and run the project. make sure to fork this repo to find future updates.
